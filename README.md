# PeteMS boxen

This is a Boxen project forked to my tastes.

```
sudo mkdir -p /opt/boxen
sudo chown ${USER}:admin /opt/boxen
git clone https://github.com/petems/our-boxen.git /opt/boxen/repo
cd /opt/boxen/repo
script/boxen
```
